# AI Website Builder

An AI-powered Website Builder developed using the MERN Stack. This application allows users to generate websites by simply entering a text prompt. The AI processes the prompt and generates a website based on the user's requirements.

## 🚀 Features
- Generate websites from AI prompts
- Responsive and user-friendly interface
- AI API integration
- MERN Stack architecture
- RESTful API implementation
- MongoDB database integration

## 🛠️ Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- JavaScript
- HTML
- CSS

## ⚙️ Installation

```bash
git clone https://github.com/Priyankakumari3118/WebsiteBuilder.git
cd WebsiteBuilder
npm install
npm start
```
